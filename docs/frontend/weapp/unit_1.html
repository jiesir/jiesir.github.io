# 微信小程序页面与h5通讯

标题

通过小程序webview组件进行与h5页面的通讯

<web-view src="{{url}}" bindload="bindload" binderror="binderror" bindmessage="bindmessage"></web-view>

//网页加载成功时候触发此事件。e.detail = { src }
bindload(){
	console.log('webview load success')
}

//网页加载失败的时候触发此事件。e.detail = { src }
binderror(){
	console.log('webview load error')
}

//网页向小程序 postMessage 时，会在特定时机（小程序后退、组件销毁、分享）触发并收到消息。e.detail = { data }，data是多次 postMessage 的参数组成的数组
bindmessage(e){
	console.log(e.detail.data)
}

# 在h5代码中引入JSSDK

<script type="text/javascript" src="https://res.wx.qq.com/open/js/jweixin-1.4.0.js" defer></script>

# API

# 获取当前环境

wx.miniProgram.getEnv(function(res) { console.log(res.miniprogram) })

注意

返回true为小程序环境,否则为false

# 向小程序发送消息，会在特定时机（小程序后退、组件销毁、分享）触发组件的message事件

wx.miniProgram.postMessage({ data: 'info' })
wx.miniProgram.postMessage({ data: {obj: 'info'} })

# h5页面跳转到小程序

wx.miniProgram.navigateTo({url: '/pages/common/index'})

# h5页面跳转到小程序并传参(重要)

const params = {a:1,b:2}
wx.miniProgram.navigateTo({ url:`/pages/common/index?params=${JSON.stringify(params)}`})

# 小程序跳转h5传值

url='https://www.baidu.com?token=djkja222ffa113'

信息

将url放入webview的src中，h5端截取url参数
